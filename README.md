                                                        Turnero Web – Clínica (Proyecto Integrador)
Autores: Arias, Ailín - Paniagua, José
---------------------------------------------------------
Descripción
Este proyecto fue desarrollado como Trabajo Práctico Integrador (TPI) para la materia Programación III de la Universidad Tecnológica Nacional. Consiste en un sistema de turnos web para una clínica ficticia, que permite gestionar la solicitud y visualización de turnos de manera simple e intuitiva.

El objetivo principal fue aplicar los conocimientos de desarrollo frontend utilizando tecnologías base, enfocándonos en la organización del código, la manipulación del DOM y la experiencia del usuario.
---------------------------------------------------------
Funcionalidades

- Solicitud de turnos con fecha y horario.

- Selección de especialidad y/o profesional (según implementación).

- Validación de formularios con JavaScript.

- Visualización dinámica de turnos asignados.

- Cancelación o eliminación de turnos (si fue implementado).

- Interfaz clara y responsive (según alcance del proyecto).
---------------------------------------------------------
Tecnologías utilizadas:

- HTML5 → Estructura del sistema.

- CSS3 → Estilos, diseño y maquetado.

- JavaScript (Vanilla JS) → Lógica del sistema, validaciones y manipulación del DOM.
---------------------------------------------------------
Objetivos del proyecto:

- Aplicar conceptos de programación orientada a objeto.

- Practicar manipulación del DOM.

- Implementar validaciones de datos en el frontend.

- Simular un flujo real de gestión de turnos.
